HTML

  • The head element contains information about the webpage.
    1. The head includes metadata which is the information about accessibility, search engines, and performance.
    2. For metadata tags, there are title, style, base, link, meta, and script.
  • The body element represents the visible content shown to the user.
    1. In the body, there are tags used to structure the content.
    2. h1, h2, h3, h4, h5, and h6 are used as headers.
    3. p is used as paragraph or a block of text.
    4. ol, ul, and li are used for lists.
    5. img is used for image with image attributes.
    6. br is used for line of empty space.
    7. a is used for anchor which created link to the webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add the starter code to the current working branch
  • git commit -m "add starter code": commit is created for staged changes
  • git pull origin main: make sure you have the latest version of the project
  • git push origin branch name: push the changes we have locally to our remote branch
